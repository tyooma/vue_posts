# Vue-posts

## Performed:
- login/logout form
- SPA with lists of posts
- user's profile page
- comments to each post

## Used technologies:
- HTML5
- CSS3 + BEM Methodology, Materialize
- Vue.js, Vuex, Vue-Router, Axios

## API
- https://jsonplaceholder.typicode.com/

## Demo
- [DEMO](https://tyooma.github.io/vue_posts/)

## Run
- use 'npm run serve' to run
